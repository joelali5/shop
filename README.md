## ✨ Features

### User Features
- **Product Browsing**: View all products with responsive grid layout
- **Category Filtering**: Browse products by categories (Clothes, Electronics, Furniture, Shoes, Miscellaneous)
- **Product Details**: View detailed information about individual products
- **Shopping Cart**: Add/remove items, update quantities, view total
- **User Authentication**: Sign up and login with Supabase authentication
- **Protected Routes**: Checkout page accessible only to authenticated users
- **Responsive Design**: Fully responsive across mobile, tablet, and desktop devices

### Technical Features
- **Optimized Images**: Automatic fallback images with error handling
- **Consistent Card Layouts**: Equal height product cards with proper content alignment
- **Loading States**: Smooth loading indicators for async operations
- **Error Handling**: Comprehensive error boundaries and user-friendly messages
- **Data Prefetching**: React Router loaders for optimal performance

## 🛠 Tech Stack

### Frontend
- **React 18** - UI library
- **React Router DOM** - Client-side routing
- **Redux Toolkit** - State management (cart functionality)
- **React Hook Form** - Form handling and validation
- **Tailwind CSS** - Styling and responsive design

### Backend & Services
- **Supabase** - Authentication and user management
- **Platzi Fake Store API** - Product data API

### Development Tools
- **Vite** - Build tool and dev server
- **ESLint** - Code linting
- **React Hot Toast** - Toast notifications
